## 📖 About The Project

**ASPIRE** is a modern, responsive website for a study abroad consultancy that helps students achieve their dreams of international education. The platform showcases services, popular study destinations, courses, testimonials, and provides an intuitive way for prospective students to connect with advisors.

### ✨ Key Features

- 🌍 **Study Abroad Hotspots** - Explore top destinations including USA, UK, Canada, Australia, Ireland, and New Zealand
- 📚 **Course Spotlight** - Browse popular courses and programs
- 👥 **Meet the Founders** - Learn about the team behind ASPIRE
- 📊 **Statistics Dashboard** - Real-time success metrics and student outcomes
- 🎓 **University Partnerships** - Ticker showcasing partner universities worldwide
- 💬 **Testimonials** - Success stories from students who achieved their dreams
- 📰 **News & Updates** - Latest updates in international education
- 📷 **Instagram Integration** - Social media presence and community
- 📱 **Fully Responsive** - Optimized for all devices

## 🚀 Getting Started

### Prerequisites

Make sure you have Node.js (v18+) and npm installed on your system.

```bash
# Check Node.js version
node --version

# Check npm version
npm --version
```

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/piyush-nirmal/ASPIRE_demo1.git
   cd ASPIRE_demo1
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` to view the application.
